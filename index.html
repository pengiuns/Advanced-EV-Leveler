Introduction

The Advanced EV-Trainer is your solution for a constant EV-Training without a single change for different EV-Trainings. This script combines all existent EV-Trainings in one huge, user configured solution !

How to use the EV-Trainer

Once setup the script does everything by itself. You can edit the config.lua to your needs.

Example:

Region = "Kanto" -- "Kanto" / "Johto" / "Hoenn"
autoEvolve = "off" -- "on" / "off"

Hp_Training         = {"Snorlax"}
Atk_Training        = {}
Def_Training        = {"Snorlax"}
Spd_Training        = {"Pikachu", "Gardevoir"}
SpAtk_Training      = {"Pikachu", "Gardevoir"}
SpDef_Training      = {}
